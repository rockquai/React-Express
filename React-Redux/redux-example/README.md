### Redux
- `store` React.js 프로젝트에서 사용하는 모든 동적 데이터들을 담아두는 곳.
- `action` 어떤 변화가 일어나야 할 지 나타내는 객체.
- `reducer`  action 객체를 받았을 때, 데이터를 어떻게 바꿀지 처리할지 정의하는 객체.
